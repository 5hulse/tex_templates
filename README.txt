To use:
1) Copy beamer_template.tex and the theme_figures directory to a desired location:
  $ cp -r beamer_template.tex theme_figures <path>/<to>/<destination>
2) Rename beamer_template.tex to <whatever_you_want>.tex:
  $ mv beamer_template.tex <whatever>.tex
3) If you wish to add Figures, create a separate figures directory
  $ mkdir <path>/<to>/<destination>/figures
4) Compile using pdflatex
  $ pdflatex <whatever>.tex

Enjoy!
